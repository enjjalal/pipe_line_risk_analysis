# Input Data Organization

This directory contains organized input data for the UK Geospatial Analysis project.

## Directory Structure

- `landcover/`: Land cover data in GeoTIFF format
- `pipelines/`: Pipeline network data in GeoJSON format
- `sensors/`: Sensor readings in CSV format
- `soil/`: Soil data (currently empty)
- `terrain/`: Terrain elevation data in GeoTIFF format
- `visibles/`: Visualization outputs and reports

## Data Sources

- Land cover data: UK Centre for Ecology & Hydrology (LCM2023)
- Pipeline data: Synthetically generated realistic UK pipeline network
- Sensor data: Synthetically generated sensor readings
- Terrain data: SRTM elevation data
